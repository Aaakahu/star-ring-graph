# 星环图谱 (Star Ring Graph)

Obsidian 插件 —— 把你的知识体系画成**可视化图谱**：支持鱼骨主干和星环扇区两种布局模式，按层级展示根节点、章节、知识点、碎片、错题。点击任意节点即可高亮其关联链路，一眼看清"这道错题考的是什么知识点、属于哪个体系"。

## 功能

### 布局模式
- **鱼骨主干布局**：章节横向铺开，MoC 按真实树深度分行、前序横向铺开，碎片/错题围父节点成星团
- **星环扇区布局**：按真实树形递归分扇区，标签长度参与角度加权，半径自适应补足弧长，彻底防标签压盖
- **密度自适应**：根据每层节点数量和标签长度自动计算半径/间距，避免节点拥挤和文字重叠

### 交互功能
- **溯源高亮**：悬停节点高亮其完整关联链路，其余节点暗化
- **按住 Ctrl 锁定高亮**：按住 Ctrl 键时冻结当前高亮状态，方便对比查看
- **右键拖动画布**：右键拖拽平移画布，不干扰左键点击操作
- **Alt+右键拖动松手**：聚焦视野中心最近的 MOC 节点
- **Alt+点击 MOC**：进入聚焦模式，该 MOC 的碎片/错题星云按 Frag 内环/Error 外环同心铺开
- **Esc 退出聚焦**：按 Esc 键或点击空白处退出聚焦模式，还原原始布局
- **滚轮缩放**：支持滚轮缩放，缩放时自动调整细节显示
- **点击节点**：自动居中放大到该节点，延迟 220ms 后打开对应笔记
- **双击节点**：折叠/展开该节点的所有后代（含连线同步隐藏）
- **空白处点击**：聚焦模式下退出，否则重置视角到全局视图

### 视觉呈现
- **按层级着色**：根节点金色、章节蓝色、知识点青色、碎片/错题暗红
- **智能文字显示**：根据缩放级别和节点层级自动显示/隐藏文字标签
- **长标题换行**：超长标题自动换行显示，避免文字重叠
- **细节显现**：放大超过阈值时才显示碎片/错题星团（可配置），聚焦模式下只显示当前 MOC 的星云
- **小地图导航**：右下角小地图方便在大图谱中快速定位
- **发光效果**：激活节点和边带有发光效果
- **容器自适应**：面板拖动或窗口缩放时自动调整画布尺寸，避免黑边

## 数据来源

适配 [StudyGuide](https://github.com/) 插件的 vault 结构：

| 类型 | 目录 | 说明 |
|---|---|---|
| 思维树 | `1_高等数学Moc/` | MOC/章节/知识点，层级靠文件名前缀编码（如 `2.1.1.2.4_标题.md`） |
| Flash 卡片 | `StudyGuide/Flash/` | 碎片知识，有 frontmatter（type/topic） |
| Error 错题 | `StudyGuide/Errors/` | 错题笔记，有 frontmatter（type/topic） |

所有路径可在插件设置中修改。

## 安装

### 手动安装
1. 下载 `main.js`、`manifest.json`、`styles.css`
2. 放到 `<vault>/.obsidian/plugins/star-ring-graph/`
3. Obsidian → 设置 → 第三方插件 → 启用「星环图谱」

### 从源码构建
```bash
git clone https://github.com/Aaakahu/star-ring-graph.git
cd star-ring-graph
npm install
npm run build
```

## 使用

1. 命令面板（Ctrl+P）→ `打开星环图谱`
2. 图谱在右侧面板打开，自动加载数据
3. 悬停任意节点查看溯源链路
4. 滚轮缩放，Ctrl + 拖拽平移

## 配置

设置 → 星环图谱：

| 设置项 | 默认值 | 说明 |
|---|---|---|
| 思维树根目录 | `1_高等数学Moc` | MOC 笔记所在文件夹 |
| Flash 卡片目录 | `StudyGuide/Flash` | 碎片卡片文件夹 |
| Error 错题目录 | `StudyGuide/Errors` | 错题文件夹 |
| 章节前缀 | `2` | 要显示的章节文件名前缀 |
| 根节点名称 | `极限与无穷小` | 图谱中心节点的显示名 |
| 图谱布局 | `鱼骨主干` | 鱼骨主干（章节横向铺开）/ 星环扇区（同心圆环） |
| 细节显现缩放 | `1.5` | 放大超过该倍数时浮现碎片/错题星团（1.0–3.0） |

## 技术栈

- **AntV G6 v4.8.24**：专业图可视化库，提供径向布局和高性能渲染
- **TypeScript + esbuild**：源码编译打包
- **Obsidian Plugin API**：数据读取 + View 集成
